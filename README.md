# Project Summoner

[Link to the app](https://project-summoner.herokuapp.com/)

## Description
A League of Legends stats app capable of viewing a player's recent match history.
This includes:
- Champion played
- Champion level
- Summoner spells
- Rune path
- Keystone rune
- Duration of game
- Player's KDA
- Player's items
- Players played with and against

## Technologies Used
- League of Legends API
- LeagueJS
- React
- Node.js
- React Materialize
